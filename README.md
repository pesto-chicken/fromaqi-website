# FROMAQI 웹사이트

프로마치(FROMAQI) 샌드위치 전문점의 공식 웹사이트입니다.

## 기술 스택

- HTML5
- CSS3
- Google Fonts (Noto Sans KR)

## 주요 기능

- 반응형 웹 디자인
- 모던한 UI/UX
- 메뉴 소개
- 위치 정보
- 매장 소개

## 설치 방법

1. 저장소를 클론합니다:
```bash
git clone https://github.com/your-username/fromaqi-website.git
```

2. 프로젝트 디렉토리로 이동:
```bash
cd fromaqi-website
```

3. 웹 브라우저에서 `index.html` 파일을 엽니다.
